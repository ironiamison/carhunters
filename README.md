# Carhunters

A car spotting bounty marketplace — post rewards for rare car spots, hunt legendary machines, get paid for your eagle eyes.

Inspired by bounty platforms like pump.fun GO, but built specifically for car enthusiasts with proper categorization, verification, and car-culture UX.

## Features

- **Bounty Board** — Browse, search, and filter active bounties by category, rarity, reward, and location
- **Post Bounties** — Lock rewards in escrow with detailed car specs and submission requirements
- **Spot Submissions** — Submit geo-tagged photos with verification workflow
- **Leaderboard** — Top spotters ranked by earnings and reputation
- **Car-Specific** — Categories for JDM, hypercars, classics, muscle, European, and more

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- Lucide Icons
